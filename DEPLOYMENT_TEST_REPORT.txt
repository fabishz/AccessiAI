1. LOCAL DEPLOYMENT (Development/Testing)

   $ cd accessiai
   $ pip install -r requirements.txt
   $ streamlit run app.py
   
   Access at: http://localhost:8501

2. STREAMLIT CLOUD DEPLOYMENT (Recommended for Quick Deployment)
   
   - Push code to GitHub
   - Go to https://streamlit.io/cloud
   - Click "New app" and select repository
   - Deploy

3. DOCKER DEPLOYMENT
   
   $ docker build -t accessiai:latest .
   $ docker run -p 8501:8501 accessiai:latest
   
   Access at: http://localhost:8501

4. AWS EC2 DEPLOYMENT
   
   - Launch Ubuntu 20.04 LTS instance (t3.medium or larger)
   - Install Python 3.8+
   - Clone repository
   - Install dependencies: pip install -r requirements.txt
   - Run: streamlit run app.py --server.port=8501 --server.address=0.0.0.0
   - Configure security groups to allow port 8501

5. HEROKU DEPLOYMENT
   
   - Create Procfile with: web: streamlit run app.py --server.port=$PORT
   - $ heroku create your-app-name
   - $ git push heroku main

See DEPLOYMENT_GUIDE.md for detailed instructions for each option.
